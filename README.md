# Github Search

Github Search is a single-page application that allows users to search for Github users and sort their repos alphabetically or by stargazers count.

A live demo of the app can be found at [vanilla-github-search.netlify.app](https://vanilla-github-search.netlify.app/).

## Install

```bash
yarn install
```

## Usage

```bash
yarn start
```

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Run Tests

```bash
yarn test
```

Launches the test runner in the interactive watch mode.\
See the section about [running tests](https://facebook.github.io/create-react-app/docs/running-tests) for more information.
